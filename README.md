[![Open in Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://gitpod.io/#https://github.com/andrewmcodes/bridgetown-plausible-starter)

## Install

```sh
bundle install && yarn install
```

## Usage

```sh
yarn start
```
